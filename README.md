# Programação Web
Este repositório contém uma coleção de projetos de desenvolvimento _web_ para a cadeira de Programação Web e Mobile.

## Projetos

- **Cadastro de Usuário**: Um formulário para registro de novos usuários.
- **Minha Rotina - HTML**: Uma apresentação da rotina diária utilizando apenas HTML.
- **Minha Rotina + CSS**: Uma versão aprimorada da rotina, agora estilizada com CSS.
- **Jogo da Senha**: Um jogo interativo baseado em [Bulls and cows](https://en.wikipedia.org/wiki/Bulls_and_cows).
- **Currículo**: Uma página demonstrando o seu currículo.
- **Portfólio**: Um portfólio com os projetos realizados até agora na cadeira.

## Como Utilizar?

Você pode acessar através do [link](https://progamacao-web-mobile.vercel.app/).
